- GNU compiler

### Usage

- Problem 1

```bash
cd Problem2
# Create a python virtual enviroment
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

python haar.py --input input.png --output output.png --depth 2

deactivate
```

- Problem 2

```bash
cd Problem2
make
```
